FAQ — Rebel Pool

What is Rebel Pool?

Rebel Pool is an open-source, anti-VC liquid staking protocol.
No pre-sale, no outside investors, just code, contracts, and 100% of yield going to stakers (except for infracosts).

      How do gasless transactions work?

      (Advanced) Sign a permit off-chain, a relayer submits the transaction for you. Powered by EIP-2612 and EIP-2771. No wallet gas required if using a relay.

      Is it safe?

      All contracts are upgradeable, secured with strict access control, reentrancy guards, and are 100% open source. The UI shows current owner/admin and pause status. The truth is in the code.

      Where do fees go?

      Fee parameters are transparent and on-chain. All rewards go to stakers minus a visible, on-chain fee (settable only by the contract owner). No VC drain. Ever.

      What happens if the contract is paused?

      All actions are temporarily disabled for safety. Only the owner can pause/unpause. Admin actions are hidden for non-owners.

      Where can I verify contracts and code?

      All contract code and UI are on GitHub (Rebel Pool org and repo). You can view, or audit. The Truth Page shows current deployment addresses and links to code.

      What if I want to run my own validator?

      Pool staking to validators will be supported when Monad opens up. Validator addresses can be updated/upgraded via the contract owner.
